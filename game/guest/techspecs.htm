Technology Specifications

This web application utilizes the following technologies:

  • Gentoo Linux - Virtualization Host
  • Debian (9) Stretch - Guest running the site
  • HaProxy - SSL and load balancing proxy
  • Nginx
  • PHP / PHP-FPM
  • HTML5
  • MySQL
  • QEMU
  • Amazon Web Services (AWS) Lightsail - Public-facing Server

All additional technologies seen in this site are built in-house.
